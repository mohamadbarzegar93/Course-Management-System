# Course-Management-System
This code is a console-based university course registration and management system using C++ OOP capabilities. It implements key object-oriented programming principles such as inheritance, encapsulation, and polymorphism through the interaction between students, lecturers, and courses.

Main Features:

Person, Student, and Lecturer classes built with inheritance
Course creation and enrollment system
Validation for course capacity and duplicate enrollment
Basic data management for course availability and participant lists
Demonstrates use of constructors, class composition, static members, and data encapsulation
Menu-driven user interface for course registration and data output
